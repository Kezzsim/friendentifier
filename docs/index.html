We're training a neural network to identify the Friends from
Kemono Friends

How it works:

The neural network will look at many images of friends, sorted into folders by name and learn to make associations. This can be used to do things like:

  • Input a friend's image, get their name useful for identifying friends!
  • Identify any image as consisting of friends another network knew only of basic objects, thus it identified Keanu Reeves as 85% sweater, 15% golf ball. What percentages of which friends would his image consist of?
  • Experimental features The same data set can be used for visual convolution algorithms like the famous Deep Dream

How can I help?

We need pictures, pictures of friends! Specifically, we need them cropped such that they contain only ONE friend. Then all you have to do is upload your images to our shared Google Drive link below into the appropriate folder.

May require sign-in with a google account to add images.
If a folder doesn't exist for a friend, please make it! Different in art styles are encouraged!
